# Config username
```git
git config --global user.name "" 
```
# Config email
```
git config --global user.email "" 
```
# List all configs
```
git config -l
```

# List a specific config
```
git config <configName>
```

# Initialize git repo
```
git init
```

# Add a file to staging
```
git add . // Adds all
git add <fileName>
```

# Commit
```
git commit -m "first commit"
```
